# Custos Chrome extension

The extension adds a compact Custos price and sales-rank history panel to Amazon.com product pages.

## Load it unpacked

1. Start the local Custos backend (by default, the extension connects to `http://localhost:4400`).
2. Open `chrome://extensions` in Chrome.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select this `extension/` directory.

Use the extension's **Options** page to change the backend URL. The configured backend must be running and reachable from Chrome; the extension makes no other network requests.
